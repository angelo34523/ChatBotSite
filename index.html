Sobre a NovaIA

A NovaIA é uma inteligência artificial voltada para programadores, desenvolvida pelo time NovaIAs. Ela pode ser encontrada em diversas plataformas, como:

  • Github
  • Telegram
  • Discord
  • Gitlab

A NovaIA no Github e Gitlab funciona como um repositório enorme, onde a IA cria novos códigos periodicamente. Já no Telegram e Discord, ela atua como um ChatBot com o qual você pode conversar sobre programação.
