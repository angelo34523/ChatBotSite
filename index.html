A NovaIA é uma IA disponível em diversas plataformas, incluindo:

  • Github
  • Telegram

A NovaIA foi criada com o objetivo de ser uma IA focada em programação, atendendo a programadores e usuários interessados em programação.

No Github e Gitlab, a NovaIA mantém um repositório enorme, onde ela cria novos códigos regularmente. Já no Telegram e Discord, ela funciona como um ChatBot com o qual você pode conversar sobre programação!
